# Enclosure

3D printed case for the LilyGO T-Display S3 (version without header pins).

## Current model

- **Type**: USB-powered, no battery, no mount
- **Fits**: LilyGO T-Display S3 (170x20x8mm PCB)
- **Buttons**: accessible from outside
- **USB-C**: open slot at the bottom

## Print settings

- **Filament**: PLA or PETG
- **Layer height**: 0.2mm
- **Infill**: 15%
- **Supports**: no
- **Estimated time**: ~3h05

## Files

| File | Description |
|------|-------------|
| `stl/plugnsat-case_top.stl` | Top shell |
| `stl/plugnsat-case_bottom.stl` | Bottom shell |
